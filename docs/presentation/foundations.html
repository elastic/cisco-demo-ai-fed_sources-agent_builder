Three retrieval primitives — and hybrid is how Cisco NOCs stop swivel-chairing across Meraki, IOS-XE, and internal wikis.

KEYWORD BM25 · exact match

TAC codes, CLI strings, CVE IDs. Lexical precision when the engineer already knows the phrase.

SEMANTIC ELSER · vector

Natural language intent. “AP keeps going offline” finds the Meraki runbook even without the exact title.

HYBRID RRF · fused rank

Reciprocal Rank Fusion merges both lists. Best of precision and meaning — one result set for the NOC.

Keyword catches the code. Semantic catches the meaning. Hybrid is what you prove live on cisco-network-kb in Module 1.
